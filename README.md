## Usage

**Install**
```
npm install
```

**Start the application in dev mode with hot-module-replacement**
```
npm start
```

**Run tests**
```
npm test
```

**Build for production**
```
npm run build
```

**Run production version**
```
npm run build:start
```

**Run code style check with standard**
```
npm run check-style
```

**Generate code coverage report**
```
npm run test:cov
```
